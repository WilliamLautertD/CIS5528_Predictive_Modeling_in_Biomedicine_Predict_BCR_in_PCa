# CIS5528_Predictive_Modeling_in_Biomedicine_Predict_BCR_-in_PCa
Repository for CIS5528 project class. Our goal is to predict BCR in prostate cancer patients using multimodal data, including H&amp;E-stained whole slide images, mpMRI, and clinical information, based on CHIMERA Challenge (https://chimera.grand-challenge.org).
